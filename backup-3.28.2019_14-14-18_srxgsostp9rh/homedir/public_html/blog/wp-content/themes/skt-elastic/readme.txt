== Description ==

Elastic comes with in built page builder which has around 30 pre defined blocks and is multipurpose, lightweight, flexible, scalable, simple, modern, and can be used for cafe, restaurant, coaching, adventure, travels, resort, hotel, spa, fitness, gym, yoga, IT company, software, digitial, online business, enterprises and agency.
